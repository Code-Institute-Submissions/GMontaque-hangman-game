Aim of the game: Hangman is a simple word guessing game, where you need to guess the word to win the game

How to Play: Before the game starts you need to decide what length of word you wish to guess, the options range from a short six letter word to a more difficult nine letter word. Once you've decided, then you can start the game. To start the game, you can either press a letter on your keyboard or use the in-game letters on the screen which display the full alphabet. Once you have guessed a letter you will see that the colour has changed so it cannot be clicked or used again till you restart. If you have guessed a correct letter, it gets added to a row of blank squares which represents the word that you have to guess.

Restart the game: To restart the game simply hit the restart button

Forgotten the rules: no worries, these can be accessed by clicking on the gear symbol

Game Music: for some extra tension and immersion you can find a volume button on the top left which will play the game theme tune
